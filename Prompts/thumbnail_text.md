# YouTube Thumbnail Text Generation

## Goal
Create short, high-impact thumbnail text to increase click-through rate
for internship-focused YouTube videos.

## Prompt Used
Generate 5 YouTube thumbnail text ideas.

Rules:
- 2 to 4 words only
- Bold and emotional
- Student-focused
- High curiosity

## Sample Outputs (AI Generated)
- INTERNSHIP SECRETS
- STOP APPLYING LIKE THIS
- THIS ACTUALLY WORKS
- REJECTION TO SUCCESS
- AI CHANGED EVERYTHING

## Learning
Short, emotionally driven prompts improve viewer attention and clicks.
