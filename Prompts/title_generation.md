# YouTube Title Generation – Internship-Focused Content

## Goal
Generate high-engagement YouTube titles using insights from
audience analysis.

## Context
Target audience: Engineering students struggling to get internships.
Pain points and motivations are derived from audience_analysis.md.

## Prompt Used
Using the audience insights below, generate 10 high-click-rate
YouTube video titles for internship-focused content.

Rules:
- Maximum 60 characters
- Student-friendly language
- Curiosity + emotional hook
- Honest and relatable
